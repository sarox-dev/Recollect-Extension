Recollect

Save highlighted text as markdown.

How to use

  • Context Menu: Select text on any website, right-click, and choose "Save highlight to Recollect"
  • Keyboard Shortcut: Select text and press Ctrl+Shift+S (or Cmd+Shift+S on Mac)

What gets saved

Your highlights are saved as markdown files with metadata including:

  • Source URL and domain
  • Page title
  • Date and timestamp
  • The highlighted text

Save Location

Files are saved to your Downloads folder by default. When you save a highlight, you'll be prompted to choose a location.
