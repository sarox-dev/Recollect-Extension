Recollect Settings

Configure how Recollect captures and saves content

Recollect API

The URL where Recollect Core receives captured content. Default: http://localhost:5000/api/capture

Capture Shortcut

Format: Modifier+Key (e.g., Alt+Shift+R, Ctrl+Shift+S, Alt+R).
Browser-level shortcut can be changed at chrome://extensions/shortcuts

Auto-Save

When enabled, selecting text with the mouse will automatically save it to Recollect (after a short delay)

Markdown Export

If set, captures will also be saved as .md files in this directory. The Recollect server must have write access. Leave empty to disable.

About

Recollect Extension v1.1.0 — MVP release.
Sends captured content to your local Recollect Core API.
All data stays on your machine.
